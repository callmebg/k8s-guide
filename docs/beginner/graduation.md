# 🎓 恭喜毕业！

你已经完成了初学者轨道的全部 **30 篇**课程——从第一个 Pod 到完整微服务部署。

## 你学到的技能

### 📦 基础入门（01-10）— 把应用跑起来

- ✅ 在本地用 Kind 搭建 K8s 集群
- ✅ 编写和部署 Pod、Deployment、Service
- ✅ 使用 ConfigMap 和 Secret 管理配置
- ✅ 配置扩缩容和滚动更新
- ✅ 理解 K8s 网络（Service/Ingress/DNS）基础
- ✅ 使用 PV/PVC 配置持久化存储

### 🔧 生产化（11-20）— 把应用跑稳

- ✅ 使用 Namespace + ResourceQuota + LimitRange 做多租户隔离
- ✅ 配置 liveness/readiness/startup 探针做健康检查
- ✅ 设置 requests/limits 控制资源使用
- ✅ 用 RBAC（SA/Role/ClusterRole）做权限管理
- ✅ 管理 DaemonSet 和 StatefulSet 工作负载
- ✅ 用 Job/CronJob 处理批处理任务
- ✅ 用 Helm 管理应用包
- ✅ 部署 metrics-server + 基础监控
- ✅ 系统化排障（describe/logs/events/debug）
- ✅ 理解 Gateway API 替代 Ingress 的趋势

### 🚀 进阶实战（21-30）— 应对复杂场景

- ✅ Init Container + Sidecar 多容器模式
- ✅ NetworkPolicy 网络隔离（前后端分层）
- ✅ Ingress TLS/rewrite/灰度/限流 生产配置
- ✅ SecurityContext 加固（非 root/只读/capabilities）
- ✅ PDB 保证节点维护时的高可用
- ✅ ServiceAccount Token 认证全链路
- ✅ CRD 自定义资源扩展 K8s API
- ✅ Kustomize base + overlay 多环境管理
- ✅ 集群升级 + etcd 备份 + 证书管理
- ✅ 完整微服务部署（前端+后端+DB+缓存+Ingress+HPA+监控）
